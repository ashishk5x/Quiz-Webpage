📖 About the Project

This project is a simple accessibility-focused quiz form that demonstrates how to build forms using semantic HTML and accessible design principles.

Users can:

Enter their personal information
Answer HTML multiple-choice questions
Respond to a CSS-related question
Submit the completed quiz

The project emphasizes accessibility by using semantic elements, proper labels, fieldsets, legends, ARIA attributes, and screen-reader-friendly content.

✨ Features
Responsive layout
Semantic HTML5 structure
Accessible form design
Navigation with smooth scrolling
HTML and CSS quiz sections
Student information form
Screen-reader-only text support
Clean and organized CSS styling
Mobile-friendly design

♿ Accessibility Features

This project follows several accessibility best practices, including:

Semantic HTML elements (header, main, section, footer)
Proper use of label elements
fieldset and legend for grouping form controls
ARIA attributes (role and aria-labelledby)
Screen-reader-only text using the .sr-only utility class
Responsive typography and layout
Smooth scrolling that respects the user's reduced motion preference
